# 村田鉄筋㈱ 取付数量分析アプリ

## プロジェクト概要
- **アプリ名**: 村田鉄筋㈱ 取付数量分析アプリ
- **目的**: 元請・現場ごとに、現場で取り付けた鉄筋数量・人工数を記録し、現場別の合計数量・合計人工数・1人工あたりの取付数量を分析する業務アプリ
- **技術スタック**: Hono + TypeScript + Cloudflare Pages + Cloudflare D1 (SQLite) + TailwindCSS + Chart.js

## 主な機能（実装済み）

### 1. ダッシュボード `/` (タブ: ダッシュボード)
- 期間（開始・終了）で絞り込み
- **総取付数量**（合計 kg）
- **総人工数**（合計 人工）
- **全体の1人工あたり取付数量** （= 総取付数量 ÷ 総人工数, kg/人工）
- **元請別 取付数量ランキング**（横棒グラフ）
- **現場別 取付数量ランキング**（横棒グラフ）
- **部位別 取付数量**（横棒グラフ）
- **人員別 取付数量**（横棒グラフ・作業員数で按分計算）

### 2. 取付数量入力画面（タブ: 入力）
入力できる項目：
- **日付** (必須)
- **元請** (必須 / 過去入力からの選択可能 / datalist による補完)
- **現場名** (必須 / 過去入力からの選択可能 / datalist による補完)
- **部位** (必須 / マスタから選択)
- **数量 (kg)** (必須 / 小数可)
- **人員数 (人工)** (必須 / 0.5, 1.5 等小数可)
- **人員名** (複数登録可能 / 「追加」ボタンまたは Enter で追加 / チップ表示 / 過去入力から選択可能)
- **備考** (任意)

スマホ最適化：iOS でズームしない 16px フォント、タッチしやすい大型ボタン。

### 3. 元請・現場別分析画面（タブ: 現場別分析）
- 絞り込み：**期間 / 元請 / 現場名**
- サマリカード：合計数量・合計人工数・1人工あたり取付数量
- **元請・現場ごとのサマリ表**：取付日数, 合計数量, 合計人工数, 1人工あたり (kg/人工)
- **取付明細表**：日付・元請・現場・部位・数量・人員数・人員名
- **CSV出力**（明細＋サマリ、Excel対応BOM付き）

### 4. 一覧画面（タブ: 一覧）
- 入力済みデータ一覧表示
- **検索**（元請/現場/部位/人員名 を横断キーワード検索）
- **編集**：1クリックで入力フォームへ転送、その場で更新
- **削除**：確認ダイアログ付き
- **CSV出力**

### 5. 部位マスタ設定（タブ: 部位設定）
- 部位の **追加 / 名前変更 / 削除**
- 既存実績の部位文字列はそのまま残るため安全

- ✅ CSV出力（一覧画面 / 分析画面 両方）

## 計算ロジック
| 項目 | 計算式 |
| --- | --- |
| 合計数量 | 入力された数量 (kg) の合計 |
| 合計人工数 | 入力された人員数 (人工) の合計 |
| 1人工あたりの取付数量 | 合計数量 ÷ 合計人工数 |
| 人員別取付数量 | 各取付の数量 ÷ その日の作業員数 を作業員に按分 |

| GET | `/api/suggestions` | 元請 / 現場名 / 人員名 の選択候補 |
| GET | `/api/installations` | 取付実績検索（`from`/`to`/`contractor`/`site`/`part`/`worker`/`search`） |
| GET | `/api/installations/:id` | 取付実績1件取得 |
| POST | `/api/installations` | 取付実績登録 |
| PUT | `/api/installations/:id` | 取付実績更新 |
| DELETE | `/api/installations/:id` | 取付実績削除 |
| GET | `/api/analytics/dashboard` | ダッシュボード集計（同フィルタ可） |

## データモデル

### `installations` テーブル（取付実績）
| カラム | 型 | 説明 |
| --- | --- | --- |
| id | INTEGER PK | 自動採番 |
| work_date | TEXT | 取付日 (YYYY-MM-DD) |
| contractor | TEXT | 元請会社名 |
| site_name | TEXT | 現場名 |
| part | TEXT | 部位 |
| quantity | REAL | 取付数量 (kg, 小数可) |
| manpower | REAL | 人員数 (人工, 小数可) |
| note | TEXT | 備考 |
| created_at / updated_at | DATETIME | タイムスタンプ |

### `installation_workers` テーブル（人員）
| カラム | 型 | 説明 |
| --- | --- | --- |
| id | INTEGER PK | 自動採番 |
| installation_id | INTEGER FK | 取付実績 ID |
| worker_name | TEXT | 人員名 |

### `parts` テーブル（部位マスタ）
| カラム | 型 | 説明 |
| --- | --- | --- |
| id | INTEGER PK | 自動採番 |
| name | TEXT UNIQUE | 部位名 |
| sort_order | INTEGER | 表示順 |

初期値：基礎 / 柱 / 梁 / 壁 / スラブ / 土間 / 土木 / その他

## ユーザーガイド

1. **新規入力**：ヘッダ右上の「＋新規入力」ボタンまたは「入力」タブから入力フォームを開く
2. 日付・元請・現場名・部位・数量・人員数を入力
3. 「人員名」欄に名前を入力して「追加」（複数人OK）。チップの「×」で削除可能
4. 「登録する」で保存（過去の元請・現場・人員名はその後候補から選択可能）
5. **分析**：「現場別分析」タブで現場ごとの合計数量・人工数・1人工あたり取付数量を確認
6. **CSV出力**：分析タブ・一覧タブの「CSV」ボタンから出力（Excelで開いても文字化けしません）
7. **編集 / 削除**：「一覧」タブの各行右側のボタンから
